GitHub Bootstrap Start
======================

To setup a compatbile Jekyll basic environment installed locally.

## Installation

```bash
git clone git://github.com/davittec/bootstrap-start
cd bootstrap-start
./script/bootstrap
ls -l #list files

echo "	You can not run local service
	$ bundle exec jekyll serve

"

```
